##hubble

A Meteor site that contains sample CRUD applets, using the dataset-* meteorite packages.

===============================
#### Installation

````sh
# Should be as simple as cloning the repository...  
git clone https://github.com/awatson1978/hubble.git

# And then running it...
cd hubble
meteor
````
